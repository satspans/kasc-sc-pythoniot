Program 3

Aim

Write a Python program to find the factorial of a positive integer using a recursive function.

Algorithm

  1. Start the program.
  2. If n is 0 or 1, return 1 (base case).
  3. If n is greater than 1, call the function with n - 1.
  4. Multiply n with the result of the recursive call.
  5. Repeat the process until n becomes 1.
  6. Return all multiplied values as recursion unwinds.
  7. Display the factorial of the given number.
  8. End the program.

Python Program

def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

num = int(input("Enter a positive integer: "))
print("Factorial of", num, "is:", factorial(num))

Output

Enter a positive integer: 5
Factorial of 5 is: 120

Program 4

Aim

Write a Python program to generate the Fibonacci sequence up to a given number using a recursive function.

Algorithm

  1. Start the program.
  2. Read the input number n.
  3. Call the recursive function fibonacci_up_to(n, 0, 1).
  4. In the function, check if the current number a is greater than n.
  5. If a > n, return an empty list.
  6. Otherwise, add a to the list and call the function with b and a + b.
  7. Continue this process until all Fibonacci numbers up to n are generated.
  8. Print the final Fibonacci sequence.
  9. End the program.

Python Program

def fibonacci_up_to(n, a=0, b=1):
    if a > n:
        return []
    return [a] + fibonacci_up_to(n, b, a + b)

n = int(input("Enter a number: "))
fib_sequence = fibonacci_up_to(n)

print("Fibonacci sequence up to", n, "is:")
print(fib_sequence)

[0, 1, 1, 2, 3, 5, 8, 13, 21]

import time

# create PWM object on GPIO 18
servo = PWM(Pin(18))

# set PWM frequency to 50Hz
servo.freq(50)

def set_angle(angle):
    # convert angle (0-180) to duty cycle
    duty = int(1000 + (angle / 360) * 8000)
    servo.duty_u16(duty)

while True:
    for angle in range(0, 270, 5):
        set_angle(angle)
        time.sleep(0.05)

    for angle in range(270, -1, -5):
        set_angle(angle)

Program 14 - Smart Parking Service

Aim

To implement a Smart Parking Service using a microcontroller that monitors parking slot availability and displays vacant and occupied spots.

Algorithm

  1. Start the process.
  2. Open wokwi.com.
  3. Create a new MicroPython project using any microcontroller.
  4. Define the total number of parking spots.
  5. Read sensor data to detect whether a spot is occupied or vacant.
  6. Update the parking lot status continuously.
  7. Display the parking lot status.
  8. Find and display the nearest vacant parking spot.
  9. Repeat the process at regular intervals.
  10. End the process.

Python Program (main.py)

import random
import time

NUM_PARKING_SPOTS = 5
parking_lot = [False] * NUM_PARKING_SPOTS

def read_sensor(spot):
    return parking_lot[spot]

def update_parking_lot():
    global parking_lot
    for i in range(NUM_PARKING_SPOTS):
        parking_lot[i] = random.choice([True, False])

def display_parking_lot():
    print("Parking Lot Status:")
    for i in range(NUM_PARKING_SPOTS):
        status = "Occupied" if parking_lot[i] else "Vacant"
        print(f"Spot {i+1}: {status}")
    print()

def find_vacant_spot():
    for i in range(NUM_PARKING_SPOTS):
        if not parking_lot[i]:
            return i + 1
    return None

def main():
    print("Welcome to Smart Parking Service!")
    while True:
        update_parking_lot()
        display_parking_lot()
        vacant_spot = find_vacant_spot()

        if vacant_spot:
            print(f"Vacant parking spot available at Spot {vacant_spot}. You can park here!")
        else:
            print("Sorry, no vacant spots available.")

        time.sleep(5)

if __name__ == "__main__":
    main()

from machine import Pin

# -------------------------
# TM1637 driver (minimal)
# -------------------------
class TM1637:
    def __init__(self, clk, dio):
        self.clk = Pin(clk, Pin.OUT)
        self.dio = Pin(dio, Pin.OUT)
        self.brightness = 0x0f

    def start(self):
        self.dio.value(1)
        self.clk.value(1)
        self.dio.value(0)

    def stop(self):
        self.clk.value(0)
        self.dio.value(0)
        self.clk.value(1)
        self.dio.value(1)

    def write_byte(self, data):
        for _ in range(8):
            self.clk.value(0)
            self.dio.value(data & 1)
            data >>= 1
            self.clk.value(1)
        self.clk.value(0)
        self.clk.value(1)

    def show(self, digits):
        segs = [0x3f,0x06,0x5b,0x4f,0x66,0x6d,0x7d,0x07,
                0x7f,0x6f]
        self.start()
        self.write_byte(0x40)
        self.stop()

        self.start()
        self.write_byte(0xc0)
        for d in digits:
            self.write_byte(segs[d] if d >= 0 else 0x00)
        self.stop()

        self.start()
        self.write_byte(0x88 | self.brightness)
        self.stop()
